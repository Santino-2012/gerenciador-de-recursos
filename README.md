# Gestor Inteligente - Interface Web

Interface web moderna para o sistema de gestão de estoque, orçamento e vendas para microempresas.

## Arquivos

- `index.html` - Página principal da aplicação
- `styles.css` - Estilos CSS com tema tecnológico
- `app/main.py` - Interface Tkinter (versão desktop)
- `gestor_segurança.py` - Backend Python com lógica de negócio

## Tema

- **Tecnológico**: Cores escuras com gradientes azuis/teal
- **Simples**: Layout limpo sem poluição visual
- **IA/Tecnologia**: Design futurista com elementos de interface moderna

## Como usar

1. Abra o arquivo `index.html` em qualquer navegador moderno
2. Use os formulários para:
   - Cadastrar produtos no estoque
   - Gerar orçamentos automáticos
   - Obter dicas de vendas
3. Visualize relatórios em tempo real

## Funcionalidades

- 📦 Cadastro de produtos com estoque
- 📋 Orçamento automático baseado em área e necessidades
- 💡 Dicas de vendas personalizadas
- 📊 Relatórios visuais de estoque e projetos

## Tecnologias

- HTML5 semântico
- CSS3 com Grid e Flexbox
- Design responsivo
- Fontes Google (Inter)
- Tema dark mode
